Course Content

The focus of sophomore courses is on advanced data science and network security knowledge, enhancing students' practical skills.

Required Courses

Second-Year English

Course Description: Strengthens students' English skills, focusing on comprehensive improvement in listening, speaking, reading, and writing to prepare for academic and professional needs.

(Fall/Spring) Credits: (2/0)

Object-Oriented Programming

Course Description: Learn the concepts of object-oriented design and programming implementation, covering key knowledge such as classes, objects, inheritance, and polymorphism, cultivating object-oriented development capabilities.

(Fall/Spring) Credits: (3/0)

Data Structures

Course Description: Explore the concepts of data structures in depth, such as arrays, linked lists, stacks, queues, and tree structures, emphasizing data organization and algorithm applications.

(Fall/Spring) Credits: (0/3)

Computer Networks

Course Description: Introduce fundamental network principles and protocols, including the OSI model, TCP/IP architecture, and network security, helping students understand how network communication works.

(Fall/Spring) Credits: (0/3)

Applied Statistical Programming

Course Description: Teach statistical analysis and applications, integrating programming to conduct data analysis and modeling, cultivating practical data handling skills.

(Fall/Spring) Credits: (0/3)

Elective Courses

Organizational Behavior

Course Description: Explore the impact of individual and group behavior on organizations, covering topics like motivation, communication, leadership, and organizational culture, helping students understand behavioral factors in management practice.

(Fall/Spring) Credits: (3/0)

Marketing Management

Course Description: Introduce marketing theories and strategies, including market analysis, marketing mix, and brand management, enhancing students' ability to plan and execute market strategies.

(Fall/Spring) Credits: (3/0)

Economics

Course Description: Study the fundamental concepts of economics, analyzing how market supply and demand, price mechanisms, and government policies affect economic systems.

(Fall/Spring) Credits: (0/3)

Accounting

Course Description: Learn accounting principles and practices, developing the ability to prepare and analyze financial statements, laying the foundation for business financial management.

(Fall/Spring) Credits: (0/3)

Linear Algebra

Course Description: Introduce concepts like matrices, vector spaces, and linear transformations, applying them to solve systems of linear equations and perform data analysis.

(Fall/Spring) Credits: (0/3)

Digital Systems Design

Course Description: Explore the fundamental principles and applications of digital circuit design, covering logic gates, combinational circuits, and sequential circuits.

(Fall/Spring) Credits: (0/3)

Web Server-Side Programming

Course Description: Introduce how to develop web applications on the server side, including dynamic website construction, server setup, and database connectivity.

(Fall/Spring) Credits: (0/3)

Mobile App Programming

Course Description: Teach the development of mobile applications, including design and coding for Android and iOS platforms.

(Fall/Spring) Credits: (0/3)

Web Application Authentication

Course Description: Cover authentication techniques and best practices for web applications, learning how to design secure web authentication processes.

(Fall/Spring) Credits: (0/3)

Computer Graphics and AI Image Generation

Course Description: Introduce basic technologies of computer graphics and how to generate images using AI, including 3D modeling, image processing, and generative adversarial networks.

(Fall/Spring) Credits: (0/3)

Machine Learning Applications

Course Description: Teach the fundamental principles and practical applications of machine learning, including supervised and unsupervised learning with practical examples.

(Fall/Spring) Credits: (0/3)

Digital Logic Design

Course Description: Introduce the design and application of digital logic circuits, covering methods for analyzing and designing logical circuits.

(Fall/Spring) Credits: (3/0)

Course Content

The junior year curriculum focuses on deepening professional skills and integrating practical operations.

Required Courses

Database Systems

Course Description: Introduces the basics of database design, data models, and database management systems, helping students learn to build and manage effective database systems.

(Fall/Spring) Credits: (3/0)

Algorithms

Course Description: Delves into the design and analysis of various algorithms and introduces methods to optimize algorithm efficiency, enhancing problem-solving skills.

(Fall/Spring) Credits: (3/0)

Operating Systems

Course Description: Explores the basic principles and functionalities of operating systems, including memory management and process scheduling techniques.

(Fall/Spring) Credits: (0/3)

Graduation Project (I)

Course Description: Provides students the opportunity to conduct independent research and develop a project, aiming to integrate their learning outcomes into a complete project.

(Fall/Spring) Credits: (0/3)

Elective Courses

Financial Management

Course Description: Introduces fundamental concepts of financial management, including financial statement analysis, fund operations, and investment decisions, helping students build a foundation in financial planning and management.

(Fall/Spring) Credits: (3/0)

Web Programming

Course Description: Teaches web and network application programming skills, enabling students to use various frameworks and tools for front-end and back-end development.

(Fall/Spring) Credits: (3/0)

Computer Architecture

Course Description: Covers fundamental concepts of computer hardware architecture and system operations, including processor design and memory management.

(Fall/Spring) Credits: (3/0)

Image Processing

Course Description: Teaches basic image processing techniques and applications, including image enhancement, transformation, and analysis techniques.

(Fall/Spring) Credits: (3/0)

Game Production

Course Description: Introduces the process and design techniques of game development, allowing students to implement game projects and learn how to turn creative ideas into interactive products.

(Fall/Spring) Credits: (3/0)

Information Security Practices

Course Description: Explores information security techniques and practical operations, including the implementation of system protection and network security measures.

(Fall/Spring) Credits: (3/0)

Networking Practices (I)

Course Description: Introduces fundamental knowledge of network architecture and protocols, enabling students to learn techniques for setting up and managing small-scale network systems.

(Fall/Spring) Credits: (2/0)

AI Applications in Creativity

Course Description: Explores how artificial intelligence techniques are applied to creative and media content generation, enhancing students' innovation and practical skills.

(Fall/Spring) Credits: (3/0)

Information Security and AI Applications

Course Description: Introduces the applications of artificial intelligence in conjunction with information security, teaching students how to use AI to enhance security strategies.

(Fall/Spring) Credits: (3/0)

Human Resource Management

Course Description: Explores strategies and practices in human resource management, including recruitment, training, and performance evaluation.

(Fall/Spring) Credits: (0/3)

Database Programming

Course Description: Teaches how to write database applications, including SQL syntax and database system development techniques.

(Fall/Spring) Credits: (0/2)

System Analysis and Design

Course Description: Introduces the processes and methods of system development, teaching students how to conduct requirements analysis and system planning.

(Fall/Spring) Credits: (0/3)

Wireless Networking Technology

Course Description: Explores the fundamentals and applications of wireless networks, teaching students how to set up and manage wireless network environments.

(Fall/Spring) Credits: (0/3)

Smart IoT

Course Description: Introduces IoT technology and smart system applications, teaching students how to design and implement connected devices.

(Fall/Spring) Credits: (0/3)
